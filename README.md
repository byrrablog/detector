# 🛡️ Log Threat Detector

Python tool for analyzing logs and detecting suspicious activity, simulating SOC (Security Operations Center) workflows.

---

## 🚀 Features

* Extracts IP addresses from logs
* Detects suspicious IPs
* Identifies error patterns
* Exports findings to CSV

---

## ▶️ Usage

```bash
python detector.py
```

Input:

```text
sample_log.txt
```

---

## 📊 Example Output

```
🚨 Amenazas detectadas:
('192.168.1.100', 'IP sospechosa detectada')
('N/A', 'Error detectado en log')
```

---

## 🎯 Use Case

Simulates log analysis tasks performed by SOC analysts to detect anomalies and suspicious activity.
